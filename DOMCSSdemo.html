Manipulating DOM and CSS with JavaScript

DOM manipulation

While other demonstrations of mine have made use of DOM manipulation via JavaScript, none have used the methods specifically designed for this. createElement, appendChild, InsertBefore, removeChild are some of the methods that fulfill this purpose.

This section demonstrates createElement, appendChild, and removeChild. Once the 'Start Demo' Button has been pressed, a new div element will be appear, from which you can add and remove child nodes via the appropriately-labelled buttons. Creating at least one child node is necessary to perform the CSS manipulation demo.

CSS Manipulation

This section of the demonstration deals with implementing CSS transforms, transitions, and animations on the spawned child nodes.

Each button will generate a specific action on one of the child nodes, and will only modify specific child nodes. Thus, to achieve all 3 demonstrations, you'll need to add a minimum of 3 nodes.

Further, a button will not appear unless it has a valid target.

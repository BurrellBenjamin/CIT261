API Local Storage Demo


This page demonstrates API local storage functionality in a variety of ways.

API stands for Application Programming Interface, a powerful JavaScript tool for interacting with client-side information. One application of this is the aspect of local storage, which allows a webpage to remember a user's preferences between sessions. Previously, this was done via use of cookies, but now can be done through use of the JavaScript localStorage object. This data never expires, is never transferred to a remote server, and has a much larger size limit than cookies.


This example demonstrates a Pokemon teambuilder similar in function to Smogon University's Pokemon Showdown simulator. However rather than implementing a full imitation of Pokemon's battle mechanics, this demo is simply concerned with demonstrating teambuilding.

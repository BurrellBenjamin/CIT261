JSON-Stringify and Parse

The JSON Stringify function converts a JavaScript object into a string form to make it transmittable across the internet.

This demonstration will require you to enter data for a Pokemon Object. The property data will then be run through the JSON.stringify function, and outputted to the screen.

The JSON Parse function is the inverse of Stringify. It takes a stringified JavaScript Object and reformats it into the normal Javascript object format.

This demonstration uses the stringified object from the last demonstration (i.e.-you have to complete that demonstration first) and parses it back into standard JSON format. The button to activate this demonstration will not show up until the stringify demonstration has been run.

AJAX

AJAX stands for Asynchronous Javascript And XML. It is a way to update a webpage while it is still loaded, and to request JSON.

This demo navigates an XML tree that holds information about various Pokemon
